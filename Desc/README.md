Во-вторых, мы попытались проанализировать отзывы S7 в Яндекс Картах. 

Анализ проводился на основе этих данных - https://yandex.ru/maps/org/s7_airlines/223695870710/reviews/?ll=37.899186%2C55.414077&z=17

Вывод из всего вышеизложенного - проблема с потерей багажа существует.

Наше предложение по решению.

Для того чтобы минимизировать вероятность ошибки, предлагаем использовать систему на основе распознования изображений. Пусть весь путь багажа будет сопровождаться камерами, а система будет отслеживать его, и в случае "схода с маршрута" будет бить тревогу.
